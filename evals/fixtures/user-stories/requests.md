# Raw stakeholder requests — parcel lockers

1. As a developer, I want an index on the pickups table so partner
   payment reports stop timing out.

2. Recipients must be sent a push notification through Firebase when
   a courier deposits their parcel.

3. As the company, we want to make more money from retail partners.

4. Handle everything about parcels that aren't collected in time —
   returns, refunds, notifying everyone involved, partner adjustments,
   depot restocking, and reporting.
